# ShadowForest - Open Source C++ SDL2 game

Open Source Game created with C++ and SDL2

## How to play
* Spacebar to start level
* Arrow keys moves player
* Space bar to attack
* Destroy monsters before they move off screen

## Setup

### Linux

```
./setup.sh
```

### MacOS

```
./setup-mac.sh
```

### Build
```
cd src
make all
```

### Start
```
./ShadowForest
